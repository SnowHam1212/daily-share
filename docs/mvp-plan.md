# MVP 実装計画

## 画面レイアウト

### 認証フロー（ヘッダーなし）

```
┌─────────────────────┐
│                     │
│   ログイン /        │
│   新規登録フォーム   │
│                     │
└─────────────────────┘
```

### メイン画面（ログイン後）

```
┌─────────────────────────────┐
│  Daily Share  📅  🗺️   👤  │  ← Header（アプリ名 + タブ + ユーザーメニュー）
├─────────────────────────────┤
│                             │
│   カレンダー or マップ       │  ← コンテンツエリア
│                             │
└─────────────────────────────┘
```

- BottomNav は使わず、Header にタブを統合する
- 👤 ボタンからログアウト

---

## 画面遷移フロー

```
ログイン / 新規登録
      ↓
チームセットアップ（初回のみ）
      ↓
メイン画面
  ├── 📅 カレンダー
  │       └── [+] → 予定追加モーダル
  └── 🗺️  マップ
```
